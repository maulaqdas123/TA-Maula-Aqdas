# TA-Maula-Aqdas
Dipyo Tracker adalah sistem pemantauan bus kampus Universitas Diponegoro  berbasis web  yang dibangun menggunakan bahasa pemrograman PHP, Tujuan utamanya adalah memberikan informasi mengenai lokasi bus, kapasitas penumpang, serta memungkinkan pengguna untuk menandai kursi mereka saat naik (Tap In) dan turun (Tap Out).
